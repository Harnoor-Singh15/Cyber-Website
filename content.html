Cybersecurity Tools

Nmap (Network Mapper)

Nmap is a network scanning tool used to discover hosts, open ports, services, and vulnerabilities. It’s a must-have for reconnaissance in cybersecurity.

nmap 192.168.1.1

Scans a single IP address for open ports.

nmap -sS 192.168.1.1

Performs a stealth SYN scan (used to avoid detection).

nmap -A 192.168.1.1

Performs an aggressive scan (OS detection, version, and script scan).

nmap -p 1-65535 192.168.1.1

Scans all 65,535 ports on the target host.

nmap -sV -sC 192.168.1.1

Detects service versions and runs default scripts on the target.

Burp Suite (Community Edition)

Burp Suite is an integrated platform for web application security testing. The community edition is perfect for beginners to analyze requests and responses.

Proxy → Intercept → ON

Captures HTTP/S requests for manual inspection and modification.

Target → Site Map

Displays the complete structure of the target website for mapping vulnerabilities.

Repeater Tool

Allows you to resend modified requests to test different inputs and responses.

Intruder (Community Mode)

Used to send multiple payloads automatically (limited speed in free version).

Decoder & Comparer

Helps to decode, encode, and compare different pieces of data during testing.
